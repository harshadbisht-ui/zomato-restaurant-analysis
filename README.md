# 🍽️ Zomato Restaurant Analysis | SQL & Power BI

## 📊 Project Overview
This project analyzes Zomato restaurant data to uncover key insights about restaurant distribution, customer preferences, and market trends. The analysis was performed using SQL, and an interactive dashboard was built using Power BI.

---

## 🎯 Objectives
- Analyze the number of restaurants across cities and countries  
- Understand customer preferences (online delivery & table booking)  
- Explore cuisine distribution and popularity  
- Study restaurant ratings and trends  
- Perform price segmentation analysis  

---


## 📊 Dashboard Features
- KPI Cards (Total Restaurants, Avg Rating, Votes, Countries, Cities)  
- City-wise restaurant distribution  
- Cuisine-wise analysis  
- Online delivery & table booking insights  
- Interactive filters for dynamic exploration  

---

## 💡 Key Insights
- New Delhi has the highest number of restaurants  
- Majority of restaurants fall in mid-range pricing categories  
- A few cuisines dominate the restaurant landscape  

---

## 🛠️ Tools & Technologies
- SQL (MySQL) – Data Analysis  
- Power BI – Data Visualization & Dashboarding
